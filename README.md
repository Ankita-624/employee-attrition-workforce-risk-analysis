# Employee Attrition & Workforce Risk Analysis Dashboard

## Overview
This project focuses on analyzing employee attrition patterns and workforce risk factors using an interactive Excel dashboard. The objective was to identify operational trends affecting employee retention, workforce stability, and organizational performance through KPI reporting and workforce analytics.

The dashboard analyzes employee data across departments, job roles, compensation levels, overtime workload, and tenure to uncover key attrition drivers and support data-driven decision-making.

---

## Objectives
- Analyze workforce attrition trends
- Identify high-risk employee segments
- Monitor key workforce KPIs
- Support operational and retention-related insights
- Improve reporting visibility using Excel dashboards

---

## Key Insights
- Employees working overtime showed higher attrition rates
- Early-tenure employees were more likely to leave the organization
- Lower income groups demonstrated increased workforce risk
- Certain departments experienced comparatively higher attrition patterns
